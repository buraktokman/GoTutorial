### Instructions

0. Fork, clone or download this repository.

1. Navigate to the directory.

2. Install requirements.

3. Run the script.

	```bash
	git clone https://github.com/buraktokman/GoTutorial.git
	cd GoTutorial
	go run <script_name>.go
